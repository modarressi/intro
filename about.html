Web Page Maker is an extremely easy to use utility that allows you to create your own site without even knowing HTML. You can create your site using Web Page Maker's professional templates or start from scratch by dragging and dropping images, music and video.. into a layout. During the edit process, you can easily move the elements with your mouse to anywhere on the page. Other features include thumbnail, mouse-over effects, ready-to-use Java Script effects, text link style sheet, tables, forms, iframes etc.

The program also includes built-in FTP publisher to allow you upload your site to the Internet by pressing a publish button. The pages are compatible with Internet Explorer, Netscape, Mozilla, Opera and any other browser.
