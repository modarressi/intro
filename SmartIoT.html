Course Title: Smart IoT Lab
Course Duration: 12 Weeks (can be adjusted based on semester)


Prerequisites:
  • Basic knowledge of Python programming
  • Understanding of basic electronics (sensors, motors, etc.)
  • Familiarity with IoT concepts (basic networking and communication)
  • Basic understanding of computer vision concepts

Lab Equipment:
Raspberry Pi (for IoT communication and sensor integration)
Jetson Nano/Xavier (for AI/ML processing and autonomous driving)
TPU (for accelerating machine learning tasks)
Smart Toy Car Kit (with motors, wheels, sensors, and battery)
Stereo Vision Camera (for depth perception and 3D vision)
Laser Distance Measurement Sensor (for precise obstacle distance measurement)
Various Sensors (ultrasonic, infrared)
Actuators (motor drivers, servos)
QT Framework (for UI development)
Wi-Fi/Bluetooth modules
Power supplies and batteries
Optional: 3D printer for prototyping additional parts
